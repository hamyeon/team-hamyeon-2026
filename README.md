# 🛍️ AI 가격 검증 모델 기반 빈티지 거래 플랫폼

기존 빈티지 거래 시장에서는 같은 모델이라도 상태, 연식, 사이즈, 구성품 여부에 따라 가격 차이가 크게 발생합니다.

하지만 판매자가 가격을 임의로 설정하는 경우가 많아, 구매자는 해당 가격이 적정한지 판단하기 어렵습니다.

본 프로젝트는 이러한 문제를 해결하기 위해 다음과 같은 흐름으로 설계되었습니다.

- 상품 이미지와 메타데이터 기반 유사 거래 상품 탐색
- 거래 사례 기반 적정 가격 범위 추정
- 판매자 희망가와 AI 추정가 비교를 통한 가격 검증
- 검증 결과를 활용한 신뢰 기반 매물 관리

즉, 단순한 중고거래 서비스가 아니라 **AI가 가격 판단을 보조하는 빈티지 거래 플랫폼**을 목표로 합니다.

<br />

## 📌 Problem

### 1️⃣ 가격 정보의 불균형

동일한 신발이라도 모델, 상태, 사이즈, 구성품 여부에 따라 가격 편차가 크게 발생합니다.

그러나 판매자가 가격을 임의로 설정하는 경우가 많아 구매자는 적정 시세를 판단하기 어렵습니다.

### 2️⃣ 시세 탐색 피로

구매자는 적정 시세를 파악하기 위해 여러 거래 사례를 반복적으로 비교해야 합니다.

원하는 상품을 찾기 위해 여러 플랫폼을 지속적으로 확인해야 하므로 시간과 피로도가 큽니다.

### 3️⃣ 판매글 노출의 비효율

게시글은 업로드 시점에 따라 쉽게 묻히며, 판매자는 노출을 유지하기 위해 동일한 상품을 반복 등록해야 합니다.

결과적으로 판매자와 구매자 모두 비효율을 겪게 됩니다.

<br />

## 📌 Solution

본 서비스는 **AI 가격 검증 기능을 핵심으로 하는 빈티지 신발 거래 플랫폼**입니다.

전체 흐름은 다음과 같습니다.

1. 사용자가 상품 이미지와 기본 정보를 입력합니다.
2. 시스템이 유사 거래 사례를 탐색합니다.
3. 수집된 거래 데이터를 바탕으로 적정 가격 범위를 추정합니다.
4. 판매자 희망가와 AI 추정가를 비교합니다.
5. 가격 차이가 과도한 경우 경고 또는 등록 제한 정책을 적용합니다.

이를 통해 구매자는 더 빠르게 가격 신뢰도를 판단할 수 있고,
판매자는 보다 합리적인 가격 기준으로 상품을 등록할 수 있습니다.

<br />

## 📌 Key Features

### 1️⃣ 유사 상품 탐색

상품 이미지와 텍스트 정보를 바탕으로 유사한 빈티지 신발 거래 사례를 탐색합니다.

### 2️⃣ 가격 추정

유사 거래 데이터와 상품 상태 정보를 기반으로 적정 가격 범위와 대표 추정가를 계산합니다.

### 3️⃣ 가격 검증

판매자 희망가가 AI 추정 범위를 과도하게 벗어나는 경우 가격 이상 여부를 판단하고 경고를 제공합니다.

### 4️⃣ 신뢰 기반 매물 관리

가격 검증 결과를 바탕으로 매물 신뢰도를 표시하거나, 향후 검증된 매물 중심 노출 정책으로 확장할 수 있습니다.

### 5️⃣ 멀티모달 확장 가능성

추후에는 이미지 중심 입력만으로도 브랜드, 카테고리, 모델 후보를 추론하는 멀티모달 구조로 확장할 수 있습니다.

<br />

## 📌 Expected Impact

본 프로젝트는 구매자와 판매자 모두에게 더 신뢰할 수 있는 거래 환경을 제공하는 것을 목표로 합니다.

### 1. 구매자
- **적정 시세를 더 빠르게 판단할 수 있습니다.**  
  AI가 유사 거래 사례와 상품 상태를 바탕으로 가격 범위를 제시하므로, 매물의 가격 적정성을 더 쉽게 확인할 수 있습니다.

- **시세 탐색에 드는 시간과 피로를 줄일 수 있습니다.**  
  여러 플랫폼의 거래 사례를 반복적으로 비교하지 않아도 되어, 원하는 상품을 더 효율적으로 탐색할 수 있습니다.

### 2. 판매자
- **합리적인 가격 기준으로 상품을 등록할 수 있습니다.**  
  AI 추정가를 참고해 판매 희망가를 설정할 수 있어 가격 책정에 대한 부담을 줄일 수 있습니다.

- **매물 신뢰도와 거래 성사 가능성을 높일 수 있습니다.**  
  데이터 기반 가격 검증을 통해 구매자에게 신뢰를 줄 수 있고, 보다 원활한 거래로 이어질 수 있습니다.

### 3. 플랫폼
- **신뢰 기반 거래 구조를 구축할 수 있습니다.**  
  가격 검증 결과를 활용해 단순 게시판형 거래를 넘어, 더 신뢰도 높은 매물 중심의 구조로 확장할 수 있습니다.

- **빈티지 거래 시장의 비효율을 줄이는 기반이 됩니다.**  
  가격 판단 보조 기능을 통해 정보 비대칭과 탐색 비용을 줄이고, 더 건강한 거래 환경을 만드는 데 기여할 수 있습니다.

## 📌 Data Collection Experiments

현재 저장소에는 가격 검증 모델의 입력 데이터를 확보하기 위한 **데이터 수집 테스트 코드**가 포함되어 있습니다.

이 코드는 완성된 서비스 기능이라기보다, **어떤 플랫폼에서 어떤 형태의 거래 데이터를 확보할 수 있는지 검증하기 위한 실험 코드**입니다.

### Kream

`kream-crawling/`은 크림 데이터를 수집하는 테스트 코드입니다.

검색 결과를 수집한 뒤, 그 결과를 바탕으로 상세 거래 데이터를 추가 수집하는 2단계 구조로 되어 있습니다.

- 검색 결과 수집
- 거래 이력 수집
- CSV 기반 후속 분석 가능

### eBay

`ebay-crawling/shoepricevalidator/`는 eBay 데이터를 활용한 백엔드 실험 코드입니다.

단순 수집을 넘어서 해외 시세 검색, 환율 반영, 가격 검증 API 구조까지 연결하는 프로토타입 성격을 가집니다.

- 해외 거래 시세 활용 가능성 검증
- 환율 및 검색어 처리 실험
- 가격 검증 서버 구조 실험

### Bunjang

루트의 `bunjang_crawler.py`는 번개장터 검색 결과를 수집하는 파이썬 스크립트입니다.

국내 개인 간 거래 플랫폼의 검색 결과 데이터를 빠르게 확보할 수 있는지 검증하기 위한 코드입니다.

- 검색어 기반 상품 리스트 수집
- 가격 및 상품 메타데이터 추출
- 국내 시세 표본 확보

## 📌 MVP

현재 MVP의 목표는 **사용자가 상품 정보를 입력했을 때, AI가 추천 가격 범위와 가격 검증 결과를 제공하는 최소 기능 제품**을 구현하는 것입니다.

### MVP Goal

빈티지 신발 거래에서 판매자가 입력한 가격이 적정한지 판단할 수 있도록,

거래 데이터 기반의 추천가와 검증 결과를 사용자에게 제공하는 것을 목표로 합니다.

### MVP Scope

1. 사용자가 상품 이미지와 기본 정보를 입력합니다.
2. 시스템이 유사 거래 사례를 탐색합니다.
3. 거래 데이터를 바탕으로 추천 가격 범위를 산정합니다.
4. 판매자 희망가와 AI 추천가를 비교합니다.
5. 가격 차이가 큰 경우 경고 또는 신뢰도 결과를 제공합니다.

### MVP Output

- 유사 거래 사례 목록
- 추천 가격 범위
- 대표 추천가
- 희망가와 추천가 차이
- 가격 신뢰도 또는 경고 메시지

### What MVP Proves

이 MVP가 검증하려는 것은 단순히 서비스 화면을 만드는 것이 아니라,

**가격 검증이라는 핵심 기능이 실제 사용자 판단에 도움을 줄 수 있는가**입니다.

## 📌 Future Work

현재 MVP는 가격 검증 기능에 집중하고 있으며, 이후에는 다음과 같은 방향으로 확장할 수 있습니다.

- **이미지 기반 상품 인식**: 상품 이미지를 바탕으로 브랜드, 모델, 카테고리를 자동 추론하는 멀티모달 구조로 확장할 수 있습니다.

- **상태 평가 고도화**: 텍스트와 이미지 정보를 함께 활용해 상품 상태를 더 정교하게 정량화할 수 있습니다.

- **추천가 모델 개선**: 더 많은 거래 데이터를 반영하여 추천가 산정 로직의 정확도를 높일 수 있습니다.

- **데이터 소스 확장**: Kream, 번개장터, eBay 등 국내외 시세 데이터를 통합해 더 신뢰도 높은 가격 기준을 만들 수 있습니다.

- **서비스 기능 확장**: 가격 검증 결과를 바탕으로 등록 정책, 노출 우선순위, 신뢰도 표시 기능으로 확장할 수 있습니다.

- **정시 경매 및 인사이트**: 장기적으로는 정시 경매 시스템과 연계하고, 추천가의 근거를 설명하는 인사이트 기능까지 확장할 수 있습니다.
