# Ebay Crawling

eBay 웹사이트에서 상품 정보를 수집하는 크롤링 테스트 코드입니다.

## 1️⃣ 검색 결과 수집

사용자가 입력한 키워드를 기준으로 eBay 검색 결과를 조회한 뒤, 검색 결과 목록에서 상품 정보를 수집합니다.

GET /ebay/test?keyword=adidas

예시:

http://localhost:8080/ebay/test?keyword=adidas

수집되는 주요 정보는 다음과 같습니다.

- 상품명
- 현재 가격
- 상품 상태
- 상품 이미지 URL
- 상품 상세 페이지 URL

검색 결과는 JSON 형태로 반환되며, 이후 유사 상품 비교 및 가격 검증 기능의 입력 데이터로 활용할 수 있습니다.

## 2️⃣ 활용 목적

수집한 eBay 상품 정보는 빈티지 신발 거래 플랫폼의 가격 검증 기능을 위한 기초 데이터로 활용할 수 있습니다.

특히 다음과 같은 목적에 사용할 수 있습니다.

- 유사 상품 탐색
- 상품 상태별 가격 비교
- 해외 플랫폼 가격 참고 데이터 확보
- AI 가격 검증 모델 실험용 데이터 수집

즉, 본 크롤링 코드는 단순히 상품 정보를 조회하는 기능이 아니라,  
AI 기반 가격 검증 모델을 위한 외부 거래 데이터 수집 모듈입니다.
