## 1. 프로젝트 개요

**Vintic**은 빈티지 신발 거래에서 구매자와 판매자가 적정 가격을 판단하기 어렵다는 문제를 해결하기 위한

**AI 가격 검증 모델 기반 빈티지 신발 거래 플랫폼**입니다.

사용자가 상품 이미지와 기본 정보를 입력하면, 시스템은 유사 거래 사례를 탐색하고

상품 상태와 거래 데이터를 바탕으로 추천 가격 범위와 대표 추정가를 제공합니다.

이후 판매자가 입력한 희망가가 적정 범위 안에 있는지 비교하여 가격 검증 결과를 보여줍니다.

Vintic은 단순히 중고거래를 연결하는 서비스가 아니라,

빈티지 신발 거래에서 가격 판단을 보조하는 **신뢰 기반 거래 플랫폼**을 목표로 합니다.

---

## 2. 문제: 고객과 Pain Point

### 2.1 주요 고객

Vintic의 주요 고객은 다음과 같습니다.

| 고객 | 설명 |
| --- | --- |
| 빈티지 신발 구매자 | 중고 또는 빈티지 스니커즈를 구매하려는 사용자 |
| 빈티지 신발 판매자 | 보유한 스니커즈를 중고 플랫폼에서 판매하려는 사용자 |

### 2.2 Pain Point

빈티지 신발 거래에서는 같은 모델이라도 가격 차이가 크게 발생합니다.

| 가격 차이 요인 | 설명 |
| --- | --- |
| 상품 상태 | 오염, 마모, 변색, 스크래치, 사용감 |
| 연식 | 출시 시기와 빈티지 가치 |
| 사이즈 | 수요가 높은 사이즈 여부 |
| 구성품 | 박스, 여분 끈, 보증서 등 포함 여부 |
| 희소성 | 매물 수, 한정판 여부, 수요 |
| 플랫폼 차이 | 국내외 시세 차이, 플랫폼별 거래 방식 차이 |

기존 중고거래 플랫폼에서는 판매자가 가격을 자유롭게 설정합니다.

하지만 구매자는 그 가격이 적절한지 직접 판단해야 합니다.

구매자 입장에서는 다음 문제가 있습니다.

- 판매자가 제시한 가격이 적정한지 판단하기 어렵다.
- “상태 좋음”, “실착 적음”, “S급” 같은 표현이 주관적이다.
- 실제 상품 상태가 판매글 설명과 다를 수 있다.
- KREAM, 번개장터, 중고나라, eBay 등 여러 플랫폼을 직접 비교해야 한다.
- 시세 탐색에 시간과 피로가 많이 든다.

판매자 입장에서도 다음 문제가 있습니다.

- 얼마에 올려야 적절한지 판단하기 어렵다.
- 가격을 높게 올리면 거래가 잘 되지 않는다.
- 가격을 낮게 올리면 손해를 볼 수 있다.
- 반복적인 네고 요청과 가격 문의에 피로를 느낀다.
- 매물이 묻히면 다시 끌어올리거나 가격을 수정해야 한다.

따라서 이 시장의 핵심 문제는 단순히 “상품을 찾기 어렵다”가 아니라,

**찾은 상품의 가격이 납득 가능한지 판단하기 어렵다**는 점입니다.

## 3. 해결 아이디어

Vintic은 빈티지 신발 거래에서 발생하는 가격 불신과 탐색 피로를 줄이기 위해

**AI 가격 검증 기능**을 핵심 기능으로 제공합니다.

### 3.1 서비스 흐름

1. 판매자가 상품 이미지와 기본 정보를 입력합니다.
2. 시스템이 브랜드, 모델명, 사이즈, 상태, 구성품 여부, 판매 희망가를 확인합니다.
3. AI 서버가 유사 거래 사례를 탐색합니다.
4. 유사 거래 데이터와 상품 상태 정보를 바탕으로 추천 가격 범위를 산정합니다.
5. 대표 추정가를 제공합니다.
6. 판매자 희망가와 추천 가격 범위를 비교합니다.
7. 가격이 적정 범위 안에 있는지, 과도하게 높은지 또는 낮은지 검증 결과를 제공합니다.
8. 구매자는 가격 신뢰도를 참고하고, 판매자는 합리적인 가격 책정 기준을 얻습니다.

### 3.2 핵심 해결 방식

| 기존 방식 | Vintic의 해결 방식 |
| --- | --- |
| 판매자가 가격을 임의로 입력 | AI가 유사 거래 사례 기반 추천 가격 범위 제공 |
| 구매자가 여러 플랫폼을 직접 비교 | 시스템이 유사 거래 데이터를 탐색하여 가격 판단 보조 |
| “상태 좋음” 같은 주관적 설명에 의존 | 이미지와 메타데이터 기반 상태 분석을 가격에 반영 |
| 단일 평균가 또는 최근 거래가 참고 | 상태를 고려한 가격 범위와 대표 추정가 제공 |
| 거래 전 가격 신뢰도 판단 어려움 | 판매 희망가와 추천 가격 범위 비교 결과 제공 |

Vintic은 가격을 완전히 자동으로 결정하는 서비스가 아닙니다.

구매자와 판매자가 더 합리적으로 판단할 수 있도록 돕는 **가격 검증 보조 도구**입니다.

## 4. 기술/구현

Vintic의 기술 구현 방향은 단순히 AI 기능을 붙이는 것이 아니라,  
**상품 등록 흐름 안에서 AI가 실제로 가격 판단을 보조할 수 있도록 연결하는 것**입니다.

멘토링에서 받은 피드백처럼, 기술 키워드를 나열하기보다  
각 기술이 어떤 문제를 해결하기 위해 필요한지 중심으로 구조를 정리했습니다.

---

### 4.1 구현 목표

Vintic에서 기술적으로 구현해야 하는 핵심 목표는 다음과 같습니다.

| AI 가격 검증 서버 | 상품 이미지와 메타데이터를 바탕으로 가격 범위 추정 |

### 4.3 AI 활용 방식

Vintic에서 AI는 형식적으로 포함되는 기능이 아니라,  
**빈티지 신발 거래에서 가격 판단이 어렵다는 문제를 해결하기 위한 핵심 도구**입니다.

AI는 다음 세 가지 목적에 활용됩니다.

| AI 활용 영역 | 입력 | 출력 | 활용 목적 |
| --- | --- | --- | --- |
| 상품 정보 보조 | 신발 이미지, 사용자 입력 정보 | 브랜드/모델 후보, 색상, 카테고리 후보 | 상품 등록 보조 및 유사 거래 탐색 정확도 향상 |
| 상태 분석 | 신발 이미지, 상태 설명 | 오염, 마모, 변색, 사용감 등 상태 정보 | 상품 상태를 가격 산정에 반영 |
| 가격 검증 | 상품 메타데이터, 상태 정보, 유사 거래 데이터, 판매 희망가 | 추천 가격 범위, 대표 추정가, 가격 적정성 결과 | 판매 희망가가 합리적인지 판단 |

기존에는 전통적인 이미지 분류 모델만을 고려할 수도 있었지만,  
멘토링 이후 이미지와 텍스트를 함께 이해할 수 있는 **멀티모달 AI 활용 가능성**을 중심으로 검토하고 있습니다.  
이는 신발 사진만 보는 것이 아니라, 브랜드, 모델명, 상태 설명, 유사 거래 데이터까지 함께 고려해야 하기 때문입니다.

---

### 4.4 데이터 수집 및 가격 검증 흐름

가격 검증을 위해서는 단순히 사용자가 입력한 가격만 보는 것이 아니라,  
비슷한 상품이 실제 시장에서 어떤 가격대로 거래되고 있는지 확인해야 합니다.

현재는 다음 데이터 소스를 실험하고 있습니다.

| 데이터 소스 | 활용 목적 |
| --- | --- |
| KREAM | 스니커즈 리셀 시세와 거래 이력 참고 |
| 번개장터 | 국내 개인 간 중고거래 매물 가격 참고 |
| eBay | 해외 빈티지 스니커즈 시세와 희귀 매물 가격 참고 |

가격 검증 흐름은 다음과 같습니다.

```text
상품 이미지와 메타데이터 입력
↓
유사 거래 데이터 탐색
↓
상품 상태와 거래 사례 비교
↓
추천 가격 범위 산정
↓
판매 희망가와 비교
↓
가격 적정성 결과 제공
```

## 5. MVP/배포

Vintic의 MVP는 **AI 가격 검증 기능이 포함된 빈티지 신발 거래 플랫폼 프로토타입**입니다.

사용자는 상품 이미지와 브랜드, 모델명, 사이즈, 상태, 구성품 여부, 판매 희망가를 입력합니다.

시스템은 유사 거래 사례를 탐색하고, 이를 바탕으로 추천 가격 범위와 대표 추천가를 제공합니다.

마지막으로 판매 희망가가 추천 가격 범위와 비교했을 때 적절한지 가격 검증 결과를 보여줍니다.

### MVP 화면 구성

| 화면 | 내용 |
| --- | --- |
| 상품 입력 화면 | 상품 이미지와 기본 정보 입력 |
| 추천가 결과 화면 | 유사 거래 사례, 추천 가격 범위, 대표 추천가 표시 |
| 가격 검증 화면 | 희망가와 추천가 차이, 가격 적정성 결과 표시 |

### MVP에서 검증할 것

이 MVP를 통해 검증하려는 것은 단순한 거래 화면이 아니라, **AI가 제시하는 가격 범위와 검증 결과가 사용자의 가격 판단에 실제로 도움이 되는지**입니다.

## 6. 차별성

Vintic은 단순한 중고거래 플랫폼이 아니라,

가격 판단을 보조하는 **AI 가격 검증 중심의 신뢰 보조 플랫폼**입니다.

### 6.1 경쟁제품과의 비교

| 서비스 | 강점 | 한계 |
| --- | --- | --- |
| KREAM | 정품 검수, 리셀 시세, 한정판 거래 신뢰도 | 개별 중고 상품 상태를 반영한 가격 검증은 제한적 |
| 번개장터 | 다양한 개인 간 거래, 높은 접근성 | 가격 판단 책임이 구매자와 판매자에게 전가됨 |
| eBay | 글로벌 거래 데이터, 경매 구조 | 국내 사용자 장벽이 있고, 개별 상품 상태 검증은 제한적 |
| Vintic | 유사 거래 사례와 상태 정보를 바탕으로 가격 검증 제공 | MVP 단계에서는 가격 검증 기능의 사용자 효용 검증이 우선 |

### 6.2 우리 서비스의 차별점

| 차별점 | 설명 |
| --- | --- |
| 가격 검증 중심 | 단순 시세 제공이 아니라 판매 희망가가 적절한지 검증 |
| 상태 반영 | 상품 이미지와 상태 정보를 가격 판단에 반영 |
| 가격 범위 제공 | 단일 가격이 아니라 추천 가격 범위와 대표 추정가 제공 |
| 근거 기반 판단 | 유사 거래 사례와 가격 차이를 함께 제공 |
| 구매자와 판매자 모두 지원 | 구매자에게는 가격 신뢰도, 판매자에게는 가격 책정 기준 제공 |
| 사전 검증 구조 | 거래 후 검수가 아니라 거래 전 가격과 상태 판단 근거 제공 |

기존 플랫폼들이 주로 “상품을 보여주고 거래를 연결하는 것”에 집중한다면,

Vintic은 “이 상품의 가격이 납득 가능한가”를 판단할 수 있도록 돕는 데 집중합니다.

## 7. PMF: Product-Market Fit

Vintic의 PMF는 실제 사용자가 겪는 문제에 우리 서비스가 적합한 해결책인지 검증하는 과정입니다.

우리 팀의 핵심 가설은 다음과 같습니다.

> 중고 및 빈티지 스니커즈 거래에서 사용자는 단순히 상품을 더 많이 보고 싶은 것이 아니라,
> 
> 
> 그 상품의 가격이 납득 가능한지 알고 싶어 한다.
> 
> 따라서 유사 거래 사례와 상품 상태 정보를 바탕으로 추천 가격 범위를 제공하면,
> 
> 구매자의 가격 불신과 판매자의 가격 책정 부담을 줄일 수 있다.
> 

이 가설을 검증하기 위해 MVP 단계에서는 가격 검증 기능을 핵심 기능으로 설정합니다.

검증하고자 하는 질문은 다음과 같습니다.

- 사용자는 중고 및 빈티지 신발 거래에서 가격 판단에 실제로 어려움을 느끼는가?
- AI가 제시하는 추천 가격 범위가 구매자의 가격 불신을 줄이는가?
- 판매자는 AI 가격 추천 기능을 통해 가격 책정 부담을 줄일 수 있는가?
- 상태 기반 가격 보정과 유사 거래 사례 제공이 거래 신뢰도를 높이는가?
- 가격 검증 기능이 실제 거래 의사결정에 영향을 주는가?

## 10. AI 서비스 활용 계획

산학트랙 서비스 구현 관점에서, Vintic은 API Call을 통해 AI 기능을 다음 목적에 활용하고자 합니다.

### 10.1 이미지 기반 상품 정보 자동 추출

| 항목 | 내용 |
| --- | --- |
| 목적 | 상품 등록 보조 및 유사 거래 탐색 정확도 향상 |
| 입력 | 신발 이미지, 필요 시 간단한 보조 텍스트 |
| 출력 | 브랜드명 후보, 모델명 후보, 카테고리 후보, 컬러 및 스타일 후보, 신뢰도 점수 |
| 활용 | 판매자가 상품 정보를 더 쉽게 입력하고, 시스템이 유사 거래 사례를 더 정확히 찾는 데 활용 |

### 10.2 이미지 기반 상태 분석

| 항목 | 내용 |
| --- | --- |
| 목적 | 상품 상태를 가격 검증에 반영 |
| 입력 | 신발 이미지 여러 장, 사용자가 입력한 상태 설명 |
| 출력 | 오염 여부, 마모 정도, 변색 여부, 상태 등급, 종합 상태 점수 |
| 활용 | 상태 점수를 가격 산정 과정의 감가 요소로 반영 |

### 10.3 AI 가격 검증

| 항목 | 내용 |
| --- | --- |
| 목적 | 판매 희망가의 적정성 판단 |
| 입력 | 이미지, 브랜드, 모델명, 사이즈, 상태 정보, 구성품 포함 여부, 판매 희망가, 유사 거래 데이터 |
| 출력 | 유사 거래 상품 목록, 적정 가격 범위, 대표 추정가, 판매 희망가와의 차이, 고가 또는 저가 여부 판단 결과 |
| 활용 | 구매자에게는 가격 신뢰도 판단 근거를 제공하고, 판매자에게는 합리적인 가격 책정 기준을 제공 |

## 11. 기대 효과

### 11.1 구매자 측면

구매자는 적정 시세를 더 빠르게 판단할 수 있습니다.

AI가 유사 거래 사례와 상품 상태를 바탕으로 가격 범위를 제시하므로, 매물의 가격 적정성을 더 쉽게 확인할 수 있습니다.

또한 여러 플랫폼의 거래 사례를 반복적으로 비교하지 않아도 되어,

시세 탐색에 드는 시간과 피로를 줄일 수 있습니다.

### 11.2 판매자 측면

판매자는 AI 추천가를 참고하여 보다 합리적인 가격 기준으로 상품을 등록할 수 있습니다.

가격 책정에 대한 부담을 줄이고, 구매자에게 가격의 근거를 설명하기 쉬워집니다.

가격 검증 결과가 함께 제공되면 매물 신뢰도와 거래 성사 가능성을 높이는 데에도 도움이 될 수 있습니다.

### 11.3 플랫폼 측면

플랫폼은 단순 게시판형 거래 구조를 넘어, 신뢰 기반 거래 구조로 확장할 수 있습니다.

가격 검증 결과를 활용해 검증된 매물 중심 노출, 가격 신뢰도 표시, 상태 리포트 제공 등으로 발전할 수 있습니다.

## 13. 정리

Vintic은 단순히 중고거래 플랫폼을 하나 더 만드는 프로젝트가 아닙니다.

우리 팀은 빈티지 신발 거래에서 가장 중요한 문제가

**상품을 찾는 것보다, 찾은 상품의 가격이 납득 가능한지 판단하는 것**이라고 보았습니다.

기존 플랫폼은 정품 검수, 거래 연결, 글로벌 매물 탐색 등 각각의 강점을 가지고 있지만,

개별 중고 상품의 상태와 판매자 희망가를 바탕으로 가격 적정성을 구체적으로 검증해주는 기능은 제한적입니다.

Vintic은 AI를 활용해 상품 상태를 분석하고, 유사 거래 사례를 바탕으로 추천 가격 범위와 가격 적정성 판단 근거를 제공합니다.

이를 통해 구매자는 가격 불신을 줄이고, 판매자는 가격 책정 부담을 줄일 수 있습니다.

따라서 Vintic은 거래 중개 중심 플랫폼이 아니라,

**가격 검증 중심의 신뢰 보조 플랫폼**을 목표로 합니다.
